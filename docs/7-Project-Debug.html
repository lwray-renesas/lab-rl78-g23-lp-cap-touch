7. Debugging the Project

Here we will guide you through debugging/running the RL78/G23 128p FPB low power capacitive touch project.

7.1 - The Debug Session

First ensure the project you wish to debug is active by clicking on it.
Then select the debug icon.
If presented with the "Select Configuration" dialog then select [Project_Name] HardwareDebug.

e2studio will then prompt you and ask if you would like to switch to the debug perspective, answer in the affirmative i.e. "Switch" as this will provide you with all the default tools and views used for debugging an embedded application.

When the application first runs, it will pause on entry to the start function, when it does this we need to click the Run button.
The program will then run-to and pause at the main function, we then need to click the Run button again.

Now the program is running you can touch the electrodes on the RL78/G23 128p FPB and see the on board LED's illuminate during a press.
